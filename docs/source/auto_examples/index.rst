Lightcurve Generation Examples
===============================

This directory contains example scripts demonstrating the use of hypothesis-lightcurves
for generating synthetic astronomical lightcurves.

The examples are processed by Sphinx-Gallery during documentation build to create
an interactive gallery with both code and output visualizations.
